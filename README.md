Compares two configuration files and shows a difference.

## Usage

```
  gendiff <file1> <file2> [options]
```

Options:

-V, --version        output the version number

-f, --format [type]  Output format , default = "tree"

-h, --help           output usage information

## Setup

```
make install
```
